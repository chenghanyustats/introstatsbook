Welcome

This is the website for my introductory statistics book. This book serves as a main reference book for my MATH 4720 Statistical Methods and MATH 4740 Biostatistical Methods at Marquette University.1 Some topics can also be discussed in an introductory data science course. You’ll learn basic probability and statistical concepts as well as data analysis techniques such as linear regression using R computing software. The book balances the following aspects of statistics:

  • mathematical expression and computation
  • distribution-based and simulation-based inferences
  • methodology and applications
  • classical/frequentist and Bayesian approaches
